## PASSO 1: ABRIR O GITHUB (5 minutos)

1. Abra o navegador (Chrome, Edge, etc.)
2. Digite: **https://github.com**
3. Clique em **"Sign up"** (canto superior direito)
4. Preencha:
   - Email: **seu_email@gmail.com**
   - Password: **crie uma senha forte**
   - Username: **ghf-contratos**
5. Clique em **"Create account"**
6. Abra seu email e clique no link de confirmação

---

## PASSO 2: ABRIR O RENDER.COM (5 minutos)

1. Abra o navegador
2. Digite: **https://render.com**
3. Clique em **"Get Started for Free"**
4. Clique em **"Sign up with GitHub"**
5. Autorize o acesso
6. Pronto! Conta criada

---

## PASSO 3: ENVIAR O CÓDIGO PARA O GITHUB (10 minutos)

### Opção A: GitHub Desktop (MAIS FÁCIL)

1. Baixe: **https://desktop.github.com**
2. Instale e abra
3. Faça login com sua conta do GitHub
4. Clique em **"File"** > **"Add Local Repository"**
5. Clique em **"Choose..."** e selecione a pasta:
   ```
   C:\Users\leila\OneDrive\docmentos\Documentos\New OpenCode Project\projeto-ghf
   ```
6. Clique em **"Add Repository"**
7. Clique em **"Publish Repository"**
8. Nome: **ghf-contratos**
9. Desmarque **"Keep this code private"** (para funcionar)
10. Clique em **"Publish Repository"**

---

## PASSO 4: HOSPEDAR NO RENDER.COM (5 minutos)

1. Acesse: **https://dashboard.render.com**
2. Clique em **"New"** > **"Web Service"**
3. Conecte sua conta do GitHub
4. Selecione o repositório: **ghf-contratos**
5. Preencha:
   - **Name**: ghf-contratos
   - **Runtime**: Python
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn app:app`
6. Em **"Environment"**, adicione:
   - **Key**: PYTHON_VERSION
   - **Value**: 3.11.0
7. Clique em **"Create Web Service"**
8. Aguarde 2-3 minutos para deploy

---

## PASSO 5: ACESSAR O SISTEMA

Após o deploy, você receberá uma URL como:

### 3. Não consigo fazer deploy
